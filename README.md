# AI Telecom Agent (Ollama Project)

This project demonstrates how to use [Ollama](https://ollama.com) with Python to optimize telecom costs.

## Requirements
- Python >= 3.12
- Ollama installed locally ([download here](https://ollama.com/download/windows))

## Setup
```bash
git clone https://github.com/username/ai_telecom_agent.git
cd ai_telecom_agent
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
pip install -r requirements.txt
